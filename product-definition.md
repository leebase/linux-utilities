# Product Definition

- Project: linux-utilities
- Primary language: C
- Problem statement: Linux users need small, trustworthy command-line utilities
  that do one job clearly without hidden services, telemetry, or dependency
  sprawl.
- First product: `sysdiff`, a small utility for comparing system state through
  explicit, fixture-backed behavior.
- Users: Linux power users, administrators, consultants, and developers who
  value auditable tools.
- Success criteria: strict warning-free builds, deterministic tests, clear
  command behavior, small source surface, and reviewable releases.
